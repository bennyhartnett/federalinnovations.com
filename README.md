## Development

This repository hosts a static website deployed via GitHub Pages.

### Local Development

To run the site locally, simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

Then visit `http://localhost:8000` in your browser.

### Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the main branch. The custom domain is configured via the `CNAME` file.
